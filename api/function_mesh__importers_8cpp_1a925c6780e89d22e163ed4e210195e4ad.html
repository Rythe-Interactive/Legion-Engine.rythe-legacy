template<class T>
void legion::core::detail::handleGltfBuffer(const tinygltf::Buffer &buffer, const tinygltf::BufferView &bufferView, std::vector<T> *data)¶

Function to copy tinygltf buffer data into the correct mesh data vector.

Parameters

  • buffer – - The tinygltf::Buffer buffer containing the data

  • bufferView – - the tinygltf::BufferView containing information about data size and offset

  • data – - std::Vector<T> where the buffer is going to be copied into. The vector will be resized to vector.size()+(tinygltf data size)
