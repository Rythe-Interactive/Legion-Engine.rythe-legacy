Function legion::core::detail::handleGltfVertexColor¶

Function Documentation¶

void legion::core::detail::handleGltfVertexColor(const tinygltf::Buffer &buffer, const tinygltf::BufferView &bufferView, int accessorType, int componentType, std::vector<math::color> *data)¶

Function to handle vertex color of tinygltf.

Parameters

  • buffer – - tinygltf::Buffer containing the mesh data

  • bufferView – - tinygltf::BufferView containing information about the buffer (data size/data offset)

  • accessorType – - tinygltf accessorType, Vertex color is expected to come in vec3 or vec4 - will be handled by the function

  • componentType – - tinygltf componentType, Vertex color is expected to come in float, unsigned byte or unsigned short - will be handled by the function

  • data – - std::vector<color> the destination of the data copy. The vector will be resized to vector.size()+(tinygltf vertex color size)
